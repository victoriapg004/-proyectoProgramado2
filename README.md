# -proyectoProgramado2

Sistema de Voluntariado Cruz Roja

Descripción

Este proyecto consiste en un sistema de gestión de voluntariado para la Cruz Roja. Permite registrar voluntarios, actividades y participaciones, además de generar reportes que facilitan el control y seguimiento de las actividades realizadas.

Objetivo

Facilitar la administración de los voluntarios y actividades mediante una aplicación web basada en una arquitectura por capas.

Tecnologías utilizadas

-Python
FastAPI
-SQLAlchemy
-MySQL
-HTML
-CSS
-JavaScript
-Git y GitHub

Módulos del sistema

Voluntarios

* Registrar voluntarios.
* Consultar voluntarios registrados.

Actividades

-Registrar actividades.
-Consultar actividades.

Participaciones

-Registrar participaciones.
-Controlar capacidad máxima de actividades.
-Validar estado de voluntarios.

Reportes

-Voluntario con más horas.
-Actividad con más participaciones.
-Cantidad de voluntarios activos.

Integrantes

-Victoria
-Aracelly
-Yuli

Arquitectura

El sistema fue desarrollado utilizando una arquitectura por capas:

-Entity
-Repository
-Schemas
-Service
-Controller
-Config


Descripción del proyecto

Sistema web para la gestión de voluntariado de la Cruz Roja que permite administrar voluntarios, actividades y participaciones. El sistema implementa reglas de negocio como control de capacidad máxima, validación de voluntarios activos e informes estadísticos para apoyar la toma de decisiones.
